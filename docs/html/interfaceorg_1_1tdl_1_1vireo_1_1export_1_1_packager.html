Detailed Description

Create packages suitable for depositing.

This object is responsible for translating a Vireo Submission into a single package file that fully describes the submission according to the standard it implements. This package is then deposited into a repository via the Depositor.

Generate a new package for this submission.

The package returned will file system resources so it is very important that any caller who constructs a package also calls delete() to free those resources used.
